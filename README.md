# 🍳 ChefMate – Smart Recipe Finder

ChefMate is a full-stack web application that intelligently suggests recipes based on ingredients available with the user. It enhances everyday cooking by providing dynamic recipe recommendations, detailed instructions, and a seamless cooking experience.

---

## 🚀 Key Highlights

- 🔍 Ingredient-based smart recipe recommendation  
- ⚡ Dynamic filtering based on ingredient match  
- 📖 Detailed recipe view with steps and metadata  
- 👨‍🍳 Guided step-by-step cooking mode  
- ❤️ Save and manage favorite recipes  
- ➕ Add custom ingredients  

---

## 🧱 Tech Stack

### Frontend
- React (TypeScript)
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js
